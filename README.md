# demo-k8s

Some demonstrations on Kubernetes.

## Demonstration list

* [docker-image](demos/docker-image): Build a Docker image.
* [ns](demos/ns): namespaces.
* [pods](demos/pods): Pods.
* [env-vars](demos/env-vars): Environment variables in a pod.
* [configmaps](demos/configmaps): ConfigMaps.
* [secrets](demos/secrets): Secrets.
* [labels](demos/labels): Labels.
* [deployments](demos/deployments): Deployments.
* [services](demos/services): Services.
* [updates](demos/updates): Update strategies.
* [probes](demos/probes): Probes.
* [resource-management](demos/resource-management): Requests.

## Demo environment

Built to be run on a cluster with one control plane and one node.

If you need a cluster, you can use [killercoda - Kubernetes Playground](https://killercoda.com/playgrounds/scenario/kubernetes) or [kubeadm-automation](https://github.com/vvision/kubeadm-automation).
